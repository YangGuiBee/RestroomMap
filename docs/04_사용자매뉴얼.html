1. 서비스 소개

화장실SOS는 운전 중 급하게 화장실이 필요한 상황을 위한 모바일 웹앱(PWA)입니다. 앱을 실행하면 현재 위치를 기준으로 가장 가까운 공공화장실을 즉시 보여주고, 버튼 한 번으로 내비게이션 안내까지 연결됩니다. 별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.

2. 시작하기

2.1 브라우저에서 바로 사용

1스마트폰/차량 브라우저에서 wc-sos.vercel.app 접속
2위치 권한 허용 팝업이 뜨면 "허용" 선택 (거부하면 서울시청 기준으로 대체 표시됩니다)

2.2 홈 화면에 앱처럼 설치 (선택)

브라우저 메뉴에서 "홈 화면에 추가"를 선택하면 아이콘이 생성되어, 매번 주소를 입력하지 않고 앱처럼 바로 실행할 수 있습니다. 오프라인 상태에서도 마지막으로 조회한 지역의 화장실 목록은 확인 가능합니다.

3. 화면 구성

3.1 상단 바

  • ◎ 내 위치 — 현재 GPS 위치로 재검색
  • 🔍 특정위치 — 지금 있는 곳이 아닌 다른 장소(목적지 등) 주변을 미리 검색

3.2 지도 영역

주황색 점은 검색한 특정위치, 파란색 점은 내 실제 위치입니다. 화장실 마커에는 가까운 순서대로 번호가 매겨져 있으며, 번호를 탭하면 상세정보가 열립니다.

3.3 하단 카드

가까운 화장실이 카드 형태로 나열됩니다. 첫 번째 카드는 1 빨간 배지로 강조됩니다.

버튼동작
안내 시작카카오내비 → 티맵 → 카카오맵 웹 순서로 자동 연결되어 길안내 시작
이용불가현재 시각 기준 운영시간이 아닌 것으로 판단된 곳 (그래도 탭하면 안내는 시작됩니다 — 정보가 틀렸을 수 있으니 참고용)
‹‹ / ››화면에 다 안 보이는 다음/이전 순위 화장실로 페이지 전환

4. 주요 기능

4.1 상세정보 보기

지도 위 번호 마커나 하단 카드를 탭하면 아래 정보가 담긴 화면이 아래에서 올라옵니다.

  • 개방시간(요일별 다른 경우 포함)
  • 남/녀 변기수
  • 장애인용 화장실 여부
  • 기저귀교환대 여부
  • 비상벨/CCTV 설치 여부
  • 관리기관 전화번호 — 탭하면 바로 전화 연결

4.2 특정위치 검색

지금 있는 곳이 아니라 앞으로 갈 장소(예: "강남역") 주변 화장실을 미리 확인하고 싶을 때 사용합니다. 검색 후에도 "안내 시작"은 항상 지금 내 실제 위치에서 출발합니다.

4.3 다크모드

별도 설정 없이 기기의 다크모드 설정을 자동으로 따라갑니다(야간 운전 시 눈부심 완화).

5. 자주 묻는 질문

Q. 위치 권한을 거부했는데 어떻게 되나요?
A. 서울시청을 기준으로 화장실 목록이 표시되고, 상단에 안내 배너가 뜹니다. "내 위치" 버튼을 다시 눌러 권한을 허용하면 정상 동작합니다.
Q. "이용불가"인데 정말 문을 닫았나요?
A. 공공데이터의 개방시간 텍스트를 자동으로 해석한 결과라 실제와 다를 수 있습니다. 확실하지 않은 경우는 아예 표시하지 않으므로, 표시된 "이용불가"는 어느 정도 근거가 있는 판단입니다. 그래도 정확도가 100%는 아니니 참고용으로 봐주세요.
Q. 카카오내비/티맵이 없으면 어떻게 되나요?
A. 자동으로 카카오맵 웹 길찾기 화면으로 넘어갑니다. 별도 조작이 필요 없습니다.
Q. 오프라인에서도 되나요?
A. 한 번 접속했던 지역이라면, 인터넷이 안 되는 상황에서도 마지막으로 불러온 화장실 목록은 볼 수 있습니다(경로 안내는 인터넷 필요).
⚠️ 운전 중에는 반드시 정차 후 화면을 조작해 주세요.
