# 화장실SOS(RestroomMap) 프로그램 목록

| 항목 | 내용 |
|---|---|
| 문서버전 | v1.0 |
| 작성일 | 2026-07-17 |
| 기준 커밋 | main 브랜치 최신 |

디렉토리별로 실제 소스 파일과 각 프로그램(모듈)의 개요를 정리한다. SRP(단일 책임 원칙)에 따라 파일 하나가 역할 하나만 담당하도록 구성되어 있다.

---

## 1. 진입점

| 프로그램명 | 경로 | 개요 |
|---|---|---|
| index.html | `/index.html` | HTML 진입점. 메타 태그(다크모드 theme-color, 파비콘), React 마운트 지점 |
| main.tsx | `src/main.tsx` (10줄) | React 앱 부트스트랩. `App` 컴포넌트를 `#root`에 마운트 |
| App.tsx | `src/App.tsx` (171줄) | 최상위 컴포넌트. 전체 상태(위치/검색/선택/페이지) 조율, 하위 컴포넌트 조립 |

---

## 2. 화면 컴포넌트 (`src/components/`)

| 프로그램명 | 경로 | 개요 |
|---|---|---|
| MapView.tsx | `src/components/MapView.tsx` (182줄) | 카카오맵 렌더링. 내 위치 마커, 검색 위치 마커, 화장실 마커(순위 번호·픽셀 충돌 회피) 관리. 지도 중심/줌 자동 조정(현재 카드 페이지 기준 bounds-fit) |
| BottomCards.tsx | `src/components/BottomCards.tsx` (86줄) | 하단 가까운 화장실 카드 스트립. 순위 배지, 거리, 안내시작/이용불가 버튼, 페이지 전환(‹‹/››) |
| DetailSheet.tsx | `src/components/DetailSheet.tsx` (106줄) | 마커/카드 탭 시 열리는 상세정보 바텀시트. 개방시간·시설정보·전화·안내시작 |
| LocationSearch.tsx | `src/components/LocationSearch.tsx` (91줄) | "특정위치" 검색 패널. 카카오 Places API로 장소·주소 키워드 검색 |

---

## 3. 커스텀 훅 (`src/hooks/`)

| 프로그램명 | 경로 | 개요 |
|---|---|---|
| useKakaoLoader.ts | `src/hooks/useKakaoLoader.ts` (45줄) | 카카오맵 JS SDK 동적 로드. 로딩/준비/에러/키없음 상태 반환 |
| useGeolocation.ts | `src/hooks/useGeolocation.ts` (58줄) | 브라우저 Geolocation API 래핑. 거부/타임아웃/미지원 시 서울시청 폴백 |
| useNearbyRestrooms.ts | `src/hooks/useNearbyRestrooms.ts` (49줄) | 좌표 기준 주변 타일 로드→거리 정렬. 3×3 격자에 결과 없으면 5×5로 자동 확장 |
| useIsDesktop.ts | `src/hooks/useIsDesktop.ts` (17줄) | 화면 폭(min-width 768px) 기준 데스크톱 여부 판정 — 하단 카드 개수(3/6) 분기용 |

---

## 4. 라이브러리 (`src/lib/`)

| 프로그램명 | 경로 | 개요 |
|---|---|---|
| geo.ts | `src/lib/geo.ts` (32줄) | Haversine 공식 거리 계산, 목록 정렬, 거리 표시 포맷(`formatDistance`) |
| geohash.ts | `src/lib/geohash.ts` (62줄) | 위경도 ↔ geohash 인코딩, 인접 타일(3×3/5×5) 좌표 계산 |
| tiles.ts | `src/lib/tiles.ts` (37줄) | 정적 타일 JSON 로더(인덱스·타일 메모리 캐시). 데이터 소스 추상화 계층 |
| navi.ts | `src/lib/navi.ts` (52줄) | 길안내 딥링크 체인(카카오내비→티맵→카카오맵 웹). 앱 전환 감지, 출발지 자동 채움 |
| openHours.ts | `src/lib/openHours.ts` (65줄) | 개방시간 자유서식 텍스트를 파싱해 현재 이용 가능 여부(open/closed/unknown) 판정 |

## 6. 데이터 파이프라인 스크립트 (`scripts/`)

| 프로그램명 | 경로 | 개요 |
|---|---|---|

## 7. 설정 파일

| 파일 | 개요 |
|---|---|
| `vite.config.ts` | Vite 빌드 설정, vite-plugin-pwa(매니페스트·서비스워커·오프라인 캐시 규칙) |
| `tsconfig.json` | TypeScript 컴파일 설정 |
| `package.json` | 의존성, npm 스크립트(dev/build/preview/geocode/tiles) |
| `.env` / `.env.example` | 환경변수(카카오 REST 키, VWorld 키, 카카오 JS 키) — `.env`는 커밋 금지 |
| `.gitignore` | node_modules, dist, .env, 대용량 원본 데이터, 로컬 전용 문서(제안서 등) 제외 |
