## 1. GitHub 가입 및 설정

GitHub는 이 프로젝트의 **소스코드가 저장되어 있는 곳**입니다.

- [ ] 1.1 https://github.com 접속 → 우측 상단 **Sign up** 클릭 → 이메일/비밀번호/아이디 입력해 가입
- [ ] 1.2 가입한 이메일로 인증 메일이 오면 인증 완료
- [ ] 1.3 원 담당자에게 **본인의 GitHub 아이디**를 알려주고, "RestroomMap 저장소에 Collaborator로 초대해달라" 요청
- [ ] 1.4 원 담당자가 초대하면, 가입한 이메일로 초대 메일이 옵니다 → 메일에서 **Accept invitation** 클릭
- [ ] 1.5 https://github.com/YangGuiBee/RestroomMap 접속해서 코드가 보이는지 확인 (보이면 완료)

## 3. 카카오 개발자 사이트 가입 및 설정

카카오 개발자센터는 **지도를 표시하는 데 필요한 "열쇠"(API 키)를 관리하는 곳**입니다.

- [ ] 3.1 https://developers.kakao.com 접속 → 우측 상단 **로그인** → 카카오계정 없으면 **회원가입** 진행 (카카오톡 쓰시던 계정 그대로 써도 됩니다)
- [ ] 3.2 로그인 후 **내 애플리케이션** 메뉴 클릭
- [ ] 3.3 원 담당자에게 아래 둘 중 하나를 요청:
  - (A) 본인의 카카오 계정을 `restroommap` 앱의 멤버로 초대해달라고 요청, 또는
  - (B) `.env` 파일에 들어갈 키 값(REST API 키, JavaScript 키)을 안전한 방법으로 전달받기
- [ ] 3.4 (A를 선택한 경우) 초대 알림 확인 후 수락 → **내 애플리케이션 → restroommap → 앱 설정 → 플랫폼 키**에서 키 확인 가능한지 체크
- [ ] 3.5 (B를 선택한 경우) 전달받은 키 값을 잘 보관(다음 5장에서 `.env` 파일에 입력하게 됩니다)

## 4. data.go.kr(공공데이터포털) 가입 및 설정

공공데이터포털은 **전국 공중화장실 원본 정보를 매달 새로 받아오는 곳**입니다. 평소엔 필요 없고, **데이터를 갱신할 때(월 1회 권장)**만 사용합니다.

- [ ] 4.1 https://www.data.go.kr 접속 → 우측 상단 **회원가입**
- [ ] 4.2 개인 회원으로 가입 진행(이메일 인증 필요)
- [ ] 4.3 로그인 후 검색창에 **"전국공중화장실표준데이터"** 검색
- [ ] 4.4 검색 결과에서 해당 데이터 클릭 → **다운로드** 버튼으로 CSV 파일 받기 (활용신청이 필요할 수 있음 — 화면 안내에 따라 간단히 신청)
- [ ] 4.5 다운로드한 파일명을 확인해두기 (나중에 `공중화장실정보.csv`로 이름을 맞춰서 사용하게 됩니다)

> 💡 지금 당장은 이 단계를 안 하셔도 됩니다. 실제로 데이터를 갱신해야 하는 시점(월 1회)에 다시 이 항목을 참고하세요.

## 5. (선택) VWorld 가입

VWorld는 지오코딩(주소를 좌표로 바꾸는 작업)이 실패했을 때 대신 시도하는 **보조 서비스**입니다. 현재 키가 **2027년 1월 17일까지 유효**하니 당장 급하지 않지만, 만료 전에 본인 명의로 새로 발급받는 것을 권장합니다.

- [ ] 5.1 https://www.vworld.kr 접속 → 회원가입
- [ ] 5.2 **오픈API** 메뉴에서 인증키 신청
- [ ] 5.3 발급받은 키를 잘 보관(추후 `.env`의 `VWORLD_KEY` 값을 교체할 때 사용)

## 6. 로컬 개발 환경 준비 (내 컴퓨터에서 코드 실행해보기)

### 6.1 필요 프로그램 설치
- [ ] 6.1.1 https://nodejs.org 접속 → **LTS(권장)** 버전 다운로드 및 설치 (다음/다음/설치 완료로 진행하면 됩니다)
- [ ] 6.1.2 https://git-scm.com 접속 → Git 다운로드 및 설치 (기본 설정 그대로 설치해도 무방)

### 6.2 코드 받아오기
- [ ] 6.2.1 바탕화면 등 원하는 위치에 폴더 하나 준비
- [ ] 6.2.2 그 폴더에서 마우스 오른쪽 클릭 → "Git Bash Here"(또는 터미널/명령 프롬프트 열기)
- [ ] 6.2.3 아래 명령어 입력 후 Enter
  ```
  git clone https://github.com/YangGuiBee/RestroomMap.git
  ```
- [ ] 6.2.4 `RestroomMap` 폴더가 생성되었는지 확인

### 6.3 비밀 열쇠 파일(.env) 만들기
- [ ] 6.3.1 `RestroomMap` 폴더 안에 `.env`라는 이름의 새 텍스트 파일 생성 (메모장으로 만들어도 되지만, 확장자가 `.txt`가 아니라 정확히 `.env`여야 합니다)
- [ ] 6.3.2 3장·5장에서 전달받은 키 값을 아래 형식대로 입력하고 저장
  ```
  KAKAO_REST_KEY=전달받은_REST_API_키_값
  VWORLD_KEY=전달받은_VWorld_키_값
  VITE_KAKAO_JS_KEY=전달받은_JavaScript_키_값
  ```

### 6.4 실행해보기
- [ ] 6.4.1 터미널에서 `RestroomMap` 폴더로 이동했는지 확인 후 아래 명령어 순서대로 입력
  ```
  npm install
  npm run dev
  ```
- [ ] 6.4.2 터미널에 나오는 주소(보통 `http://localhost:5173`)를 웹 브라우저에 입력해서 접속
- [ ] 6.4.3 지도와 화장실 목록이 정상적으로 보이면 **성공** — 로컬 개발 환경 준비 완료!

### 6.5 (배포까지 정상 연결되는지 최종 확인)
- [ ] 6.5.1 코드 아무 파일이나 아주 사소하게 수정(예: 글자 하나)
- [ ] 6.5.2 터미널에서 아래 명령어로 저장소에 반영
  ```
  git add .
  git commit -m "테스트 커밋"
  git push origin main
  ```
- [ ] 6.5.3 1~2분 후 https://wc-sos.vercel.app 접속해서 방금 수정한 내용이 반영되는지 확인
- [ ] 6.5.4 확인 후 원래대로 되돌리고 싶으면 원 담당자에게 문의하거나 `git revert` 사용
